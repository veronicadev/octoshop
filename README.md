# OctoShop
![alt text](https://i.ibb.co/qdvWsM4/homepage.png)

Simple Node js ecommerce
Technlogies used:
  * Node JS
  * Express JS
  * Vanilla Javascript
  * Html/Css
  * BEM methodology
  * Bootstrap
  
## Run the project

### Install project dependencies
```
npm i
```

### Start project
```
npm start
```
